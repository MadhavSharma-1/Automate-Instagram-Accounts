# Product Reels Automation

Admin panel to automate **AI-generated product (unboxing-style) Reels** with
affiliate captions, published to Instagram through the **official Instagram
Graph API**. Built with Next.js (App Router) + Postgres (Prisma).

> **Scope & compliance.** This tool is for promoting products you're an
> affiliate for, using sanctioned APIs:
> - Publishing uses the **official Instagram Graph API Content Publishing**
>   endpoints — not automation of the private app — so accounts stay in good
>   standing. Use Instagram **Business/Creator** accounts you own.
> - Product data comes from **official affiliate APIs** (Amazon PA-API,
>   AliExpress Affiliate, Temu/affiliate networks) rather than scraping, which
>   those sites prohibit and actively block.
> - Every caption automatically appends an **FTC affiliate disclosure (`#ad`)**
>   and an **AI-content label**, which are legally/platform required.
>
> An **approval queue** sits before publishing so a human signs off on each post.

## Features

- **Multi-account**: connect several IG Business/Creator accounts, each with its
  own daily post limit (e.g. 3/day).
- **Pluggable product sources**: adapter interface (`src/lib/sources`) with
  Amazon / AliExpress / Temu / manual implementations. Add a source by
  implementing one interface.
- **AI captions**: generated by the Claude API from your **editable prompt
  templates** (`{{title}}`, `{{price}}`, `{{features}}`, `{{link}}`).
- **AI video pipeline**: pluggable provider (`stub` / Shotstack / Replicate)
  that renders a vertical unboxing Reel and returns a public URL.
- **Approval queue**: review + edit caption, preview video, approve/reject.
- **Scheduler/worker**: publishes approved posts within each account's daily
  limit (standalone worker or serverless cron).

## Architecture

```
Product source adapter ─▶ Product ─▶ generatePostForProduct()
                                         ├─ renderVideo()      (src/lib/video.ts)
                                         └─ generateCaption()  (src/lib/caption.ts, Claude)
                                              ▼
                                     Post (PENDING_APPROVAL)
                                              ▼  (admin approves)
                                     Post (APPROVED)
                                              ▼  runDuePublishes() respects dailyLimit
                                     Instagram Graph API  (src/lib/instagram.ts)
                                              ▼
                                     Post (PUBLISHED)
```

## Setup

1. **Install & configure**
   ```bash
   npm install
   cp .env.example .env   # fill in DATABASE_URL, ANTHROPIC_API_KEY, etc.
   ```
2. **Database**
   ```bash
   npm run db:push           # or: npm run db:migrate
   npx tsx prisma/seed.ts    # default template + manual source
   ```
3. **Run the app**
   ```bash
   npm run dev               # admin panel at http://localhost:3000
   ```
4. **Run the publisher** (in another terminal, or use Vercel Cron via
   `vercel.json` → `/api/cron/publish`)
   ```bash
   npm run worker
   ```

## Getting the credentials

| What | Where |
|------|-------|
| Instagram `igUserId` + long-lived token | Meta app → Instagram Graph API → link a Business/Creator account on a FB Page |
| `ANTHROPIC_API_KEY` | console.anthropic.com |
| Amazon PA-API keys | Amazon Associates → Product Advertising API |
| AliExpress keys | AliExpress Open Platform / Affiliate |
| Temu | Temu affiliate program or a network (CJ/Awin) feed |
| Video provider | Shotstack or Replicate (set `VIDEO_PROVIDER`) |

## What's wired vs. what needs your keys

Fully working out of the box (with a DB + Anthropic key):
- Admin panel, multi-account, prompt templates, manual products
- Caption generation (Claude)
- Approval queue, scheduler/daily-limit logic
- Video pipeline with the `stub` provider (placeholder URL) so you can exercise
  the whole flow end-to-end

Marked **INTEGRATION POINT** in code — drop in your keys/SDK to activate:
- Amazon PA-API SigV4 request (`src/lib/sources/amazon.ts`)
- AliExpress / Temu affiliate calls (`src/lib/sources/{aliexpress,temu}.ts`)
- Real video rendering (`src/lib/video.ts` → Shotstack/Replicate)

## Highlights

Reels publish via the Graph API. To also add posts to a **Highlight**, mark a
post `isHighlight` — the Story → Highlight flow is a follow-up step on the
publisher (Stories are published similarly, then grouped into a Highlight).

## Security notes (before production)

- Replace the placeholder admin auth with real authentication.
- Encrypt `accessToken` at rest.
- Lock down `/api/cron/publish` (it checks `x-cron-secret`).
